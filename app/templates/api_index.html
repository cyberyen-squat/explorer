Get balance of address

ENDPOINT GET

/api/addressbalance/{address}

DESCRIPTION

Returns balance for a given address.

https://cyberyen.work/api/addressbalance/{address}

Response on success

      {"error": "ok", "message": "0.00000000"}

Response on failure

      {"error": "404", "message": "This address is invalid"}

Get best block hash
